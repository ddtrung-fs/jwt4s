# JWT library for Scala
[![Build Status](https://travis-ci.org/menzus/jwt4s.svg?branch=master)](https://travis-ci.org/menzus/jwt4s)

## Features
- Signs
- Verifies

- ISS check
- SUB check
- AUD check
- EXP check
- IAT check
- RFP check (https://tools.ietf.org/html/draft-bradley-oauth-jwt-encoded-state-05)

- HS256
- HS384
- HS512

- Safe digest equals

- Akka HTTP directive

- Typesafe config
